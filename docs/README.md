# 점심 메뉴 추천

## 기능 구현 목록

- [ ] 메뉴 추천 시작 메시지 출력
- [ ] 코치 이름 입력 요청

  - `입력 유효성 검사`
  - [ ] 이름 최소 2~4글자
  - [ ] 최대 2명 ~ 5명

- [ ] 코치 이름 저장
- [ ] 코치 별 못 먹는 음식 입력 요청

  - `입력 유효성 검사`
  - [ ] 최소 0개 ~ 2개

- [ ] 코치 별 음식 카테고리 랜덤 선택 pickNumberInRange(1, 5)

  - [ ] 선택된 카테고리 중복 검사 (주 최대 2회 중복 가능)
  - [ ] 추천할 수 없는 경우 === 다시 추천

- [ ] 코치 별 음식 종류 랜덤 선택 Randoms.shuffle(menus)[0];

  - [ ] 메뉴 중복 검사 (중복 불가능)

- [ ] 코치 별 추천 결과 출력

- [ ] 추천 완료 메세지 출력
  - [ ] 종료
